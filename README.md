# Student Grades Management System

Учебный проект по проектированию информационной системы учёта успеваемости студентов.

## Описание проекта

Целью проекта являлась разработка модуля информационной системы для автоматизации учёта успеваемости студентов.

В рамках проекта были выполнены анализ предметной области, формализация требований, моделирование бизнес-процессов, проектирование базы данных и разработка прототипа приложения.

## Выполненные работы

* Анализ предметной области
* Формализация функциональных требований
* Разработка сценариев использования (Use Case)
* Построение IDEF0-моделей
* Построение диаграмм потоков данных (DFD)
* Проектирование структуры базы данных
* Разработка UML-диаграмм
* Реализация прототипа приложения на C# (Windows Forms)
* Тестирование основных функций системы

## Используемые технологии

* C#
* Windows Forms
* PostgreSQL
* UML
* IDEF0
* DFD

## Структура репозитория

* `docs` — документация проекта и отчёт по курсовой работе
* `diagrams` — UML-диаграммы, IDEF0, DFD и другие модели
* `source` — исходный код приложения

## Основной функционал

* Просмотр успеваемости студентов
* Выставление оценок
* Расчёт среднего балла
* Формирование отчётов
